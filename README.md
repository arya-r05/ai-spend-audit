# AI Spend Audit Dashboard

Optimize, calculate, and audit your multi-model LLM API costs dynamically.

Built with Next.js, TypeScript, and Tailwind CSS. Deployed on Vercel.

## Live Demo
https://ai-spend-audit-vert-one.vercel.app

## What it does
Helps startups compare AI model costs across OpenAI, Anthropic, and Google.
Enter your token usage and get a real-time cost breakdown across all providers.

## Tech Stack
- Next.js + TypeScript
- Tailwind CSS
- Vercel (deployment)

## How I built it
- Day 1: Project setup, Next.js + TypeScript + Tailwind configuration
- Day 2: Built pricing data matrix, React state handlers for token inputs, cost comparison table
- Day 3: Added CSS visualization engine, input validation, UI polish and deployed to production

## Key Features
- Real-time cost calculation engine
- Multi-model comparison (OpenAI, Anthropic, Google)
- Responsive dashboard UI
- Input validation and error handling
